Nesse projeto busquei trabalhar tudo que aprendemos dentro do WorkShop
- Laços de reptição (for, while)
- Funções e Colletions
- Tratamento de erro (do: try/catch, throw)
- Print, variaveis, Nill coalescence
- Range, Condition, Enum

O escopo exigiu estar dentro do mundo de RPG, por tanto quis inovar no tema, se passando na semana de um trabalhador, podendo escolher que tipo de heroi/personagem ele quer ser:
- CLT, CTL Premium e Faria Limier (em uma nova atualização colocarei o Herdeiro)
Cada dia fazemos varias escolhas e nesse caso não é diferente, podendo escolher entre 3 portas, sendo que apenas uma não tem um monstro do cotidiano.
Para ganhar o nosso heroi deve passar por 5 portas, como se fosse 5 dias de trabalho, logo nosso final é sextar :)

Buscando analogias com a linguagem que no momento tenho um pouco de dominio a mais, percebi que Classes podem contribuir na construção de objetos,
estes que podemos utilizar no código, acabei aproveitando a oportunidade para testar tambem se funcionava como no Java, com ajuda do mentor entendi que 
para colocar um Swift case dentro de uma classe é necessário que esteja dentro de uma função.
